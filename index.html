Glosario del Primer módulo

En este glosario podrás encontrar varios conceptos que te ayudarán a sumergirte en el mundo de la programación.

CSS

Describe cómo se va a mostrar un documento en la pantalla, o cómo se va a imprimir, o cómo va a ser pronunciada la información presente a través de un dispositivo de lectura. Es decir, ofrece el control total sobre estilo y formato de sus documentos.

Especificidad

La especificidad es el modo que tiene el navegador de decidir qué regla se aplica si diversas reglas tienen selectores diferentes pero podrían aplicarse a un mismo elemento.

GIT

GIT es un sistema de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia y confiabilidad del mantenimiento de versiones.

GitHub

GitHub es una plataforma de desarrollo colaborativo para alojar proyectos utilizando el sistema de control de versiones GIT.

HTML

HTML, siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y estructurar el contenido en la web.

Ramas

Una rama la podemos ver como una versión de los archivos que se encuentran en nuestro repositorio, podemos crear tantas como queramos.

Repositorio

Un repositorio en GIT no es más que un “nuevo proyecto”

Remote

Remotes Un remote no es más que una versión de tu proyecto alojada en un servidor que no está en tu máquina local. Es la conexión entre tu local y la nube, entre git y github. Al crear un nuevo repositorio siempre se crea un remote default llamado “origin”.

Selectores

Un selector CSS es la primera parte de una regla CSS que indica al navegador qué elementos HTML se seleccionan para aplicarles dicha regla

Terminal

Es una herramienta que te permite interactuar con una computadora utilizando comandos de texto en lugar de una interfaz gráfica.
